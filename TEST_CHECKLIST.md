# 功能验证测试清单

在环境变量正确配置后，按以下清单进行上线前功能验证。

## 1. 认证与会话
- [ ] 注册：填写邮箱/密码，注册成功后自动登录或引导登录
- [ ] 登录：输入正确凭证后登录成功，错误凭证提示文案正确
- [ ] 会话持久化：刷新页面后用户状态仍在（cookie / localStorage）
- [ ] 登出：登出后跳转登录页，受保护页面不可访问

## 2. 拼团模块
- [ ] 列表页 `/groups`：数据加载正常，分页/筛选（`category`、价格区间、关键字）可用
- [ ] 详情页 `/groups/[id]`：展示拼团信息、参与人数、倒计时/状态
- [ ] 加入拼团：参与流程正常，按钮状态与库存/人数限制正确

## 3. 订单与钱包
- [ ] 订单列表：分页数据加载正常，状态过滤可用
- [ ] 下单流程：从详情页下单至支付页流程可达（如暂为占位则跳转正确）
- [ ] 钱包明细：分页加载与余额展示正确

## 4. 通用与 UI
- [ ] 顶部导航与路由：主要入口可达（首页、登录/注册、拼团、个人页）
- [ ] 表单校验：输入不合法时提示，提交按钮状态正确
- [ ] 反馈提示：成功/失败均有可见提示（AntD message/notification）

## 5. 网络与错误
- [ ] API 超时：超时后返回统一错误提示，页面不崩溃
- [ ] 401/403：未授权访问受保护路由时正确拦截
- [ ] 500：服务异常时展示错误页或提示

## 6. 兼容性与性能
- [ ] 移动端布局：主要页面在移动端良好显示
- [ ] 列表性能：长列表滚动流畅，图片懒加载可用
- [ ] 首屏时间：首页首屏加载在预期范围内

## 7. 部署后回归
- [ ] 预览环境（Preview）验证通过后再 Promote 到生产
- [ ] 生产环境变量与域名配置一致
- [ ] 基本流程与关键指标无异常（错误率、接口耗时）
